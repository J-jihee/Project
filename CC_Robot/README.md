# CC Robot (Child Care Robot)



### robosapiens : 로봇제어와 관련된 py파일
### wowwow : 안드로이드 app 파일

## 기능  

- 놀이모드
    - TV 모드 : Power on/off , channel control
    - 노래재생 모드 : 저장해둔 노래파일 재생
    - 녹음 모드 : 사용자의 목소리를 녹음 후 출력 

- 움직임 모드
    - 인사, 응원 등의 모션 출력
    - 로봇의 움직임 제어

- 홈 CCTV 모드
    - 앱을 통해 CCTV 화면 출력
    - 카메라 움직임 제어로 원하는 방향 출력 가능
